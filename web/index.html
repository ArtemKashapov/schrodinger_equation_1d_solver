Numerical methods for solving the radial Schrödinger equation.
A particle in the field of central forces.

\[-{1 \over B} {d^2\Psi(x) \over dx^2} + F(x) \Psi(x) = \epsilon\Psi(x), F_9(x) = -{1 \over x}\]

Points:

  • Set the boundaries of integration:
    x0 - left bound point;
    xN - right bound point;
    N - Amount of points.

    • Energy:

  • Set parameters for finding energy eigenvalues:
    E0 - starting point of search;
    Emax - extreme search point;
    dE - search step (an important parameter, for large values it is possible to skip solutions).

    • Parameters:

  • Configurable parameters:
    B - factor from the dimensionless Schrödinger equation;
    Maximum desired number of solutions;
    Potential number (0 - harmonic oscillator);
